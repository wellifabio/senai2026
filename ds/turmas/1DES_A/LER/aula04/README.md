# Aula04 - Digitalização do trabalho em grupos
Volte a grupo que fez em papel o trabalho da aula01 e utilizando o draw.io ilustre os requisitos funcionais com Diagramas de caso de uso e crie os wireframes.

### Identificar requisitos (Word)
Documente os requisitos utilizando Word, formate o documento segundo a ABNT com [capa, folha de rosto, resumo, sumário, introdução, capítulos, resultados e bibliografia].
- A introdção deve ter uma descrição de como o sistema funcion,quais problemas ele resolve, quais tecnologias utliza.

#### Os capítulos devem ser
- Regras de negócios
    - Numeradas de [RN001] até [RN00...]
- Requisitos funcionais
    - Numeradas de [RF001] até [RF00...]
    - Ilustrados com UML DCU (Diagramas de Caso de USO).
- Requisitos não funcionais
    - Numeradas de [NF001] até [NF00...]
- Wireframes das telas
    - Para celular e/ou Web.

#### A Conclusão ou resultados deve ter um breve orçamento
Apresentando os custos da implantação do pojeto
- Por fim coloque a **Bibliografia**

#### Descrição do trabalho feito em papel
- Escolher um dos sistemas sugeridos.
- Descrever como o sistema  funciona.
- Elencar as:
    - Regras de negócios
    - Requisitos funcionais.
    - Requisitos não funcionais.
- Desenhar um esboço das telas do sistema
