# ABNT para trabalhos acadêmicos

## Formatação
- Fonte: Times New Roman ou Arial, tamanho 12 para o texto e 10 para citações longas, notas de rodapé, legendas e fontes.
- Espaçamento: 1,5 entre linhas para o texto e simples para citações longas, notas de rodapé, legendas e fontes.
- Margens: 3 cm para as margens esquerda e superior, e 2 cm para as margens direita e inferior.
- Alinhamento: Justificado para o texto e centralizado para títulos e subtítulos.
- Numeração de páginas: As páginas devem ser numeradas a partir da introdução, mas a numeração deve começar a aparecer a partir da primeira página do texto (introdução). As páginas anteriores (capa, folha de rosto, resumo, sumário) não devem ser numeradas.

## Estrutura do trabalho
1. Capa
2. Folha de rosto
3. Resumo
4. Sumário
5. Introdução
6. Capítulos (Regras de negócios, Requisitos funcionais, Requisitos não funcionais, Wireframes)
7. Conclusão ou Resultados
8. Bibliografia
